Mārwāri E-Commerce WordPress Integration

This project has been transformed into an active WordPress Plugin to integrate directly with your website database at https://marwari-ecommorce.rpsdigitalworld.store/.

Standard Integration Mode: You do not need to host a separate raw HTML page. The storefront layout is dynamically compiled inside the WordPress engine and displayed on any page using a simple Shortcode.

How to deploy to public_html

  1. Gather your Files: You have 4 files in your local workspace:
    • marwari-ecommerce.php (The main plugin core)
    • style.css (The royal storefront design styles)
    • app.js (The dynamic WordPress REST API client)
    • README.md (Documentation reference)
  2. Create a Plugin Directory:

    Log in to your Hostinger/cPanel File Manager (or FTP) and navigate to your website's WordPress folder:

    public_html/wp-content/plugins/

    Create a new folder named marwari-ecommerce:

    public_html/wp-content/plugins/marwari-ecommerce/
  3. Upload the Files:

    Upload marwari-ecommerce.php, style.css, and app.js directly into that folder.

  4. Activate the Plugin:

    Log into your WordPress Dashboard (/wp-admin), go to the Plugins page, find Marwari E-Commerce Storefront, and click Activate.

    Autopilot DB Setup On activation, the plugin automatically creates custom MySQL tables in your database and seeds the products.
  5. Display on your Website:

    Create a new Page or post (e.g. named "Shop"), insert a Shortcode Block, write the following shortcode, and publish the page:

    [marwari_storefront]

Database Details

The plugin manages state natively on your WordPress database through two custom tables:

  • wp_marwari_products: Stores ID, Category, Name, Price, Description, Image URLs, and Badges.
  • wp_marwari_orders: Stores placed orders, items JSON payload, customer shipping info, totals, and order status.

User Accounts

Authentication integrates directly with your active WordPress User directory. You can log in using:

  • Customer: user@gmail.com / password123 (standard customer access)
  • Admin: admin@gmail.com / password123 (grants automatic access to the Administrative Dashboard block)
